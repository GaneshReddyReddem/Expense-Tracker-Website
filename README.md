# Expense-Tracker-Website

# 💰 Smart Expense Tracker

A modern web application to manage personal finances with **category-based tracking and real-time data analytics**.

---

## 🚀 Features

* ➕ Add income & expenses
* 🗂️ Category-based tracking (Food, Travel, Shopping, etc.)
* 📊 Interactive data analytics dashboard
* 💰 Real-time balance calculation
* 🧾 Transaction history
* ❌ Delete transactions
* 💾 Data persistence using LocalStorage

---

## 📊 Analytics Dashboard

* Category-wise expense distribution
* Income vs Expense comparison
* Spending insights for better financial decisions

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Chart.js
